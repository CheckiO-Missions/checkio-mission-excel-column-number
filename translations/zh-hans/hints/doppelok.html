好了，现在是时候考虑数学问题了。如果我们在一个字符串中只有一个字母--一切都很清楚，我们只是以整数的形式返回其 "值"。但如果有两个或更多的字母呢？让我们以 "AY "这两个字母为例进行分析。由于有两个字母，而第一个字母是 "A"，这意味着它的值是26，而不是1，因为它在 "Z "之后。接下来是字母 "Y"，它的值是25，所以我们把26+25加起来，我们的结果就是。 

name = "AY"
result = 0
while name:
    result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
    name = name[1:]

你所要做的就是把代码包在一个函数表达式中，并返回result 值。 

def column_number(name: str) -> int:

    result = 0
    while name:
        result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
        name = name[1:]
        
    return result

def column_number(name: str) -> int:

    total = 0
    for i, j in enumerate(name[::-1]):
        total += (ord(j) - 64) * (26 ** i)
        
    return total
