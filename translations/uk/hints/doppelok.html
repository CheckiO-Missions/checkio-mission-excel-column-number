Гаразд, тепер потрібно подумати над математикою. Якщо у нас одна літера в рядку - все зрозуміло, ми просто повертаємо її "значення" у вигляді цілого числа. Але якщо дві або більше літери? Розберемо на прикладі двох літер "AY". Оскільки літер дві, та перша літера "A", то це означає що вона матиме значення 27, а не 1, оскільки вона йде після "Z". Далі літера "Y", вона має значення 25, тож сумуємо 27 + 25 і наш результат. 

name = "AY"
result = 0
while name:
    result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
    name = name[1:]

Тобі залишилось обгорнути код у вираз функції, та повернути значення result. 

def column_number(name: str) -> int:

    result = 0
    while name:
        result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
        name = name[1:]
    return result

def column_number(name: str) -> int:
    total = 0
    for i, j in enumerate(name[::-1]):
        total += (ord(j) - 64) * (26 ** i)
    return total
