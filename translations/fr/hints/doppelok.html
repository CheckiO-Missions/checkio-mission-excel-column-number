Bon, il est temps de penser aux mathématiques. Si nous avons une lettre dans une chaîne de caractères, tout est clair, nous renvoyons simplement sa "valeur" sous la forme d'un entier. Mais s'il y a deux lettres ou plus? Prenons l'exemple des deux lettres "AY". Puisqu'il y a deux lettres, et que la première est "A", cela signifie qu'elle aura une valeur de 26, et non de 1, puisqu'elle vient après "Z". Ensuite, la lettre "Y" a une valeur de 25, nous ajoutons donc 26 + 25 et nous obtenons notre résultat. 

name = "AY"
result = 0
while name:
    result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
    name = name[1:]

Je ne veux pas abandonner, aidez-moi!

Tout ce que vous avez à faire est d'envelopper le code dans une expression de fonction et de renvoyer la valeur result. 

def column_number(name: str) -> int:

    result = 0
    while name:
        result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
        name = name[1:]
        
    return result

def column_number(name: str) -> int:

    total = 0
    for i, j in enumerate(name[::-1]):
        total += (ord(j) - 64) * (26 ** i)
