Potrzebujemy pierwszej litery, aby wyświetlić 1, a nie 65. Ponadto funkcja ord() może przyjąć tylko jedną literę. Proszę więc przewidzieć również to, w przypadku linii, której długość jest większa niż 1. Użyjemy pętli while, w której będziemy pobierać jeden element z linii, aż do jej zakończenia. 

name = "AY"
while name:
    print(ord(name[0]) - 64)
    name = name[1:]

25

Ok, teraz czas pomyśleć o matematyce. Jeśli mamy jedną literę w ciągu - wszystko jest jasne, po prostu zwracamy jej "wartość" w postaci liczby całkowitej. Ale jeśli są dwie lub więcej liter? Przeanalizujmy dwie litery "AY" jako przykład. Ponieważ są dwie litery, a pierwszą literą jest "A", oznacza to, że będzie ona miała wartość 27, a nie 1, ponieważ występuje po "Z". Następna jest litera "Y", która ma wartość 25, więc dodajemy 27 + 25 i otrzymujemy wynik. 

name = "AY"
result = 0
while name:
    result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
    name = name[1:]

Wszystko, co musi Pan zrobić, to zawinąć kod w wyrażenie funkcji i zwrócić wartość result. 

def column_number(name: str) -> int:

    result = 0
    while name:
        result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
        name = name[1:]
        
    return result

def column_number(name: str) -> int:

    total = 0
    for i, j in enumerate(name[::-1]):
        total += (ord(j) - 64) * (26 ** i)
        
    return total
