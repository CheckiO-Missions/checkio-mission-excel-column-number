Okay, now it's time to think about the math. If we have one letter in a string - everything is clear, we just return its "value" in the form of an integer. But if there are two or more letters? Let's analyze the two letters "AY" as an example. Since there are two letters, and the first letter is "A", this means that it will have a value of 27, not 1, since it comes after "Z". Next is the letter "Y", it has a value of 25, so we add 27 + 25 and our result. 

name = "AY"
result = 0
while name:
    result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
    name = name[1:]

All you have to do is wrap the code in a function expression and return the result value. 

def column_number(name: str) -> int:

    result = 0
    while name:
        result += (26 ** (len(name) - 1)) * (ord(name[0]) - 64)
        name = name[1:]
    return result

Okay, okay! That's the spirit!=) Here is a variant of the solution through the for loop. The mathematical formula is the same. 

def column_number(name: str) -> int:

    total = 0
    for i, j in enumerate(name[::-1]):
        total += (ord(j) - 64) * (26 ** i)
    return total
