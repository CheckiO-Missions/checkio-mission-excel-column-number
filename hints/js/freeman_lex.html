What should I do next?

Now while iterating you need to convert each character into its number equivalent, so "A" should become 1 (since it's the first number in this system), "B" - 2..."Z" - 26. For this purpose, use charCodeAt() method, which returns the code for character at exact index. 

name.charCodeAt(i)

You may notice, that this method returns 65 for "A". So, to get 1 you need to subtract 64. 

name.charCodeAt(i)-64
Explain me more about base here.

Let's explore a decimal (10-based) number "12" for example. To "get" to 12 from 0, you need to move all base (1..10) and two steps more (11, 12). Being more abstract, if you have a two-digit decimal number "ab", how many "ones" it includes? Its a*10 + b.

The same is here. If you have an excel title "AB", how many "ones" it includes? You need to move from "A" to "Z" (thru the whole base - 26) and two more steps ("AA", "AB"): a*26 + b (where "a", "b" - number equivalents of "A", 'B').

When you have big numbers in any base, the rule is the same actually: to get number of "ones", you should number on every position multiply with base in some power. In our case: 

res += (name.charCodeAt(i)-64)*26**...
How to find out this power?

Let's discover another example with bigger decimal number. If you have a number with digits 1 2 5, how many "ones" it includes? It's 1*10**2 + 2*10**1 + 5*10**0 = 125. With 0-base index "i" you may see, that i+power = length - 1. So power = length - 1 - i. This rule works for any base. 

res += (name.charCodeAt(i)-64)*26**(name.length-1-i)
So, how my final solution should look like?

You just need to join everything together and wrap your code into function expression.

Here is the whole solution:

function columnNumber(name: string): number {

    let res = 0;
    for (let i = 0; i < name.length; i++){
        res += (name.charCodeAt(i)-64)*26**(name.length-1-i)
    }
    return res
}

Convert string into array and using reduce function at each step multiply with base and add number equivalent for current character. 

function columnNumber(name: string): number {
    return [...name].reduce((col, c) => col * 26 + c.charCodeAt(0) - 64, 0);
    }
